

## 保底的意义

为什么我们要有保底机制?

因为大多数玩家是不会去尝试理解为什么说是 $10\%$ 概率, 却能十发不出的.

只会无能狂怒, 发帖怒斥概率造假.

这种情况下千万不要脑子发热去和玩家科普, 应该回复:

> "是是是, 您说得对"

然后默默改成十发必出, 坐等 "shut up and take my money".

---

此外把保底写出来, 可以给玩家一个期待.

让他们能有一个心理价位, 有了目标, 更好氪金.

而且抽卡是有沉没成本的, 抽了很多次没出, 会有极其强烈的挫败感, 破罐破摔, 直接导致弃游.

有了保底目标的话, 投入终究会有回报.

就算没达到保底, 也能够开脱

> "谁叫你不努力攒石头呢?"

这就是保底机制存在的意义.
